CSS

¿Qué es CSS?

CSS (Cascading Style Sheets o Hojas de Estilo en Cascada) es un lenguaje que permite definir la presentación visual de páginas web escritas en HTML. Se utiliza para aplicar colores, fuentes, márgenes, alineaciones y otros aspectos de diseño sin modificar la estructura del contenido.

CSS es clave para la separación entre contenido y presentación, lo que facilita el mantenimiento y mejora la accesibilidad de los sitios web.

Importancia en el desarrollo web

✅ Separación de contenido y diseño: Mantiene el HTML limpio y organizado.

✅ Diseños responsivos: Permite que las páginas se adapten a diferentes dispositivos.

✅ Optimización: Mejora la velocidad de carga al reducir el tamaño de archivos.

✅ Facilidad de mantenimiento: Un solo archivo CSS puede afectar múltiples páginas.

JavaScript

¿Qué es JavaScript?

JavaScript es un lenguaje de programación que permite crear contenido dinámico en las páginas web. Es utilizado para agregar interactividad, manipular elementos del DOM y mejorar la experiencia del usuario.

Es compatible con todos los navegadores modernos y se usa junto con HTML y CSS para desarrollar sitios web interactivos.

Importancia en el desarrollo web

✅ Interactividad: Permite crear animaciones, efectos y respuestas a eventos del usuario.

✅ Manipulación del DOM: Puede modificar elementos de la página sin necesidad de recargarla.

✅ Compatibilidad con APIs: Se usa para integrar mapas, datos en tiempo real y servicios externos.

✅ Popularidad y comunidad: Es uno de los lenguajes más usados con soporte en frameworks modernos.

Font-size Define el tamaño de la fuente del texto. px, em, rem, %, vw p { font-size: 16px; } Font-size

Box-shadow Añade una sombra a los bordes de un elemento. Desplazamiento X, Desplazamiento Y, Difuminado, Extensión, Color div { box-shadow: 5px 5px 10px gray; } Box-shadow
Overflow Controla cómo se maneja el contenido que se desborda de un elemento. visible, hidden, scroll, auto div { overflow: hidden; } Overflow
Z-index Define el tamaño de la fuente del texto. px, em, rem, %, vw p { font-size: 16px; } Z-index
Opacity Define el nivel de transparencia de un elemento. Valores numéricos entre 0 (completamente transparente) y 1 (opaco). div { opacity: 0.5; } Opacity

Text-align Define la alineación del texto dentro de un elemento. left, right, center, justify p { text-align: center; } Text-align
